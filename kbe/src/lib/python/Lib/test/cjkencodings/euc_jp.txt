Python の倡券は、1990 钳ごろから倡幌されています。
倡券荚の Guido van Rossum は兜伴脱のプログラミング咐胳≈ABC∽の倡券に徊裁していましたが、ABC は悸脱惧の誊弄にはあまり努していませんでした。
このため、Guido はより悸脱弄なプログラミング咐胳の倡券を倡幌し、毖柜 BBS 庶流のコメディ戎寥≈モンティ パイソン∽のファンである Guido はこの咐胳を≈Python∽と叹づけました。
このような秦肥から栏まれた Python の咐胳肋纷は、≈シンプル∽で≈浆评が推白∽という誊筛に脚爬が弥かれています。
驴くのスクリプト废咐胳ではユ〖ザの誊黎の网守拉を庭黎して咖」な怠墙を咐胳妥燎として艰り掐れる眷圭が驴いのですが、Python ではそういった井嘿供が纳裁されることはあまりありません。
咐胳极挛の怠墙は呵井嘎に病さえ、涩妥な怠墙は橙磨モジュ〖ルとして纳裁する、というのが Python のポリシ〖です。
